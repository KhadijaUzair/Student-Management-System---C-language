# Student Management System (C)

## Overview

The Student Management System is a console-based application developed in C to simplify the management of student records. This project demonstrates the use of structured programming, file handling, functions, and basic data management techniques.

This repository contains the complete source code and project report developed as part of an academic project.

---

## Features

- Add new student records
- View all student records
- Search student records
- Update existing student information
- Delete student records
- Menu-driven console interface
- Store and retrieve data using file handling

---

## Technologies Used

- C Programming Language
- File Handling
- Functions
- Structured Programming
- Dev-C++

## How to Run

1. Download or clone this repository.
2. Open the project in Dev-C++ or any C IDE.
3. Compile the `main.c` file.
4. Run the program.
5. Follow the on-screen menu to manage student records.
